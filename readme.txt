== Description ==

 A Light-Weight, Simple and Complete GDPR Consent Plugin for WordPress

== Features ==
 
* Cookie consent notice. 
* Consent management.
* Privacy laws & guidelines. 
* Legal documents.   


== Installation ==

1. Go to Plugins > Add New.
1. Search for "WPX GDPR Consent".
1. Click "Install Now".

OR

1. Download the zip file.
1. Upload the zip file via Plugins > Add New > Upload.

Activate the plugin. Look for "WP GDPR" in the admin menu.
